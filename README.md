# Everyday-Project
This repo consists of all the projects that I am building during the Everyday new project challenge.

## [Day 1: Zero-Trust Multi-Service Mesh Demo](./Day%201)
This project demonstrates a fully functional, 3-tier microservice architecture deployed on Kubernetes using Istio as a Service Mesh. It implements enterprise-grade traffic management, zero-trust security, and a complete observability stack.

## [Day 2: Self-Healing GitOps Infrastructure with ArgoCD](./Day%202)
This project demonstrates a **production-style GitOps platform** built using **Kubernetes + ArgoCD + Observability tools**. It features a **self-healing infrastructure** where the cluster state is continuously synchronized with Git, automatically reverting any manual changes. Key components include the **App-of-Apps pattern**, **Kustomize** for environment management, and a full observability stack with **Prometheus, Grafana, Loki, and Promtail**.
